# Dynchro

Dynchro: exploratory analysis and alignment of multiple complex inferred trajectories

## Getting started

See [the repository of the accompanying usecases](https://github.com/LouiseDck/dynchro_paper) for guidance on how to
use dynchro.

## Installation

Install the latest development version:

```bash
pip install git+https://github.com/LouiseDck/dynchro.git@main
```
